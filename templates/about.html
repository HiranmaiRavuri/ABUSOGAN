üöÄ System Workflow

The system workflow begins when a user uploads a video file through the web interface. The video is processed and converted into frames, which are resized and normalized before being analyzed by the trained deep learning model. Each frame is classified individually, and the final prediction is determined using majority voting across all frames. The system then displays the detected activity along with a description and recommended preventive measures, allowing users to quickly understand the situation and take appropriate action.
